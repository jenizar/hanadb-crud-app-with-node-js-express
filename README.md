Install node js module in sap bas terminal:

$>npm install express body-parser hbs @sap/hana-client --save

to run app in sap bas (npm run start), change script in package.json:

  "scripts": {
  
    "start": "node index.js"
    
  },

to run app in vs code (your local pc):

$> node index.js
